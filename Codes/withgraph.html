CRC Simulation Help

What is CRC?

Cyclic Redundancy Check (CRC) is an error-detecting code commonly used to detect accidental changes to raw data during transmission or storage.

How to Use This Simulation

  1. Click "Start New Simulation" to begin
  2. Enter your message when prompted
  3. Choose whether to inject an error
  4. Observe how different CRC types detect errors

Understanding the Visualization

  • Original Message: The data you entered
  • Received Messages: What the receiver gets after transmission
  • Error Info Panel: Shows details about any injected errors
  • Performance Metrics: Compares different CRC types

Tips

  • Try different messages to see how CRC performance varies
  • Compare error detection rates between CRC-8, CRC-16, and CRC-32
  • Use the zoom controls to examine bit patterns more closely
